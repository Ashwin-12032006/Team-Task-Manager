🚀 Team Task Manager

A simple and modern web application designed to help teams manage tasks, assign work, track progress, and improve collaboration efficiently.

✨ Features
User Authentication
Create & Assign Tasks
Update Task Status
Team Collaboration
Project Tracking
Dashboard with Task Analytics
Responsive and User-Friendly Interface
🛠️ Tech Stack
React.js
Tailwind CSS
Node.js
Express.js
MongoDB
JWT Authentication
▶️ How to Run
Install Dependencies
npm install
Start Frontend
npm start
Start Backend
npm run server
